## how to run

1. run `npm i`
2. create a `.env` file like this
```
DISCORD_TOKEN=<insert your bot token here>
DISCORD_CLIENT=<insert your client id here>
```
3. run `node deploy-commands.js`
4. run `node .` or `pm2 index.js` if you want
